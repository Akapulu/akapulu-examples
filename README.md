# Akapulu Examples

Public examples for building with Akapulu.

This repository is organized into two top-level sections:

- `fundamentals`
  - Core integration examples based on the Akapulu Web SDK.
  - Includes:
    - `next-react-ui-demo` (prebuilt UI path with `@akapulu/react-ui`)
    - `next-react-custom-ui-demo` (fully custom UI path with `@akapulu/react`)

- `example-scenarios`
  - Scenario-focused end-to-end examples.
  - Includes:
    - `patient-intake-screening`

## Repository structure

```text
akapulu-examples/
  README.md
  fundamentals/
    next-react-ui-demo/
    next-react-custom-ui-demo/
  example-scenarios/
```
